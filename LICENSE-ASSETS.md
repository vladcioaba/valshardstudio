## What this grant does NOT cover

This grant covers the **output** of the editor, not the editor itself.
The editor source code remains licensed under the terms in the
top-level `LICENSE` file (PolyForm Noncommercial 1.0.0 for editor
source; MIT for the runtime extension library under
`extensions/OCSExtension/`).

In particular: shipping a copy of the editor binary or editor source
inside your game's distribution is **not** covered by this grant. If
you want to redistribute the runtime library so that your game can
load `.csb` files at runtime, link the MIT-licensed
`extensions/OCSExtension/` static library — that is the supported
path.
